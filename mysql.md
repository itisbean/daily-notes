# Mysql

## MySQL各存储引擎的特点

*[InnoDB]: MySQL5.5.8版本后是默认数据引擎
### InnoDB 存储引擎

- 支持事务
- 行锁设计
- 支持外键
- 默认REPEATABLE级别
- 使用next-key locking的策略来避免幻读（phantom）
- 插入缓冲（insert buffer）
- 二次写（double write）
- 自适应哈希索引（adaptive hash index）
- 预读（read ahead）

### MyISAM存储引擎

- 表锁设计
- 支持全文索引
