# Daily Notes

## This repository record my daily notes , using the vscode plugin : [Foam](https://github.com/foambubble/foam)

### Foam Usage

#### link pages

`[[demo]]` , `Cmd` + `Click` ( or `f12` )

#### creating new notes

`Cmd` + `p` , `> New note` (**VS Code Markdown Notes**), *FILENAME*

#### graph visualisation

`Cmd` + `p` , `> show graph` (**Show graph**)

#### focus

`Cmd` + `k`, `z`
